OmniOS Community Edition releases r151022av & r151024x are now available.

Weekly releases for w/c 16th of April 2018.
> This update requires a reboot on r151022.

### Security fixes

* `perl` updated to fix:
  * [CVE-2018-6797](https://cve.mitre.org/cgi-bin/cvename.cgi?name=2018-6797)
  * [CVE-2018-6798](https://cve.mitre.org/cgi-bin/cvename.cgi?name=2018-6798)
  * [CVE-2018-6913](https://cve.mitre.org/cgi-bin/cvename.cgi?name=2018-6913)
* `zsh` updated to fix:
  * [CVE-2018-1100](https://cve.mitre.org/cgi-bin/cvename.cgi?name=2018-1100)
* `nghttp2` updated to fix:
  * [CVE-2018-1000168](https://cve.mitre.org/cgi-bin/cvename.cgi?name=2018-1000168)


Additionally, for r151022, an updated Xen network interface driver has been
released to fix a problem running under Xen 4.

For more details, see [https://omnios.org/releasenotes](/releasenotes.html)
